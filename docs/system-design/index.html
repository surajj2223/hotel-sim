How to read this site

Every PRD carries two independent statuses. Contract status (Draft → Frozen) governs whether the spec is a valid build target. Once frozen, a contract may not drift silently — any deviation a builder must make is recorded in the PRD's Contract-Drift Log. Until a row is arbitrated and the contract re-frozen, the code is the source of truth for that clause. Build status (Not Started → In Progress → Implemented) tracks reality as waves land. Wave 1 may begin only after frozen contracts gate it.
